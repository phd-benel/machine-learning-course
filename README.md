# Machine Learning Module (15 hours) — Course Support

This repository contains the **HTML course material** for a 15-hour Machine Learning module. It is designed as a coherent learning path that balances **conceptual rigor** (Chapters 1–2) with **engineering practice** (Chapter 3).

## How to read the course

- Open `index.html` to access the table of contents.
- Then navigate to the chapters in `pages/`.
- Figures and illustrations are stored in `Figures/`.

## Course structure (high level)

- **Chapter 1 — Foundations & paradigm shift (~1h)**  
  From *Software 1.0* (explicit rules) to *Software 2.0* (inductive programming via optimization). Key pillars: **Data**, **Compute**, and **Algorithms**. Clarifies the hierarchy between **AI**, **Machine Learning**, and **Deep Learning**.

- **Chapter 2 — The Machine Learning toolbox (~8h)**  
  A taxonomy of models and how to choose them depending on the problem:
  - **Supervised learning**: linear/logistic regression → neural networks  
  - **Unsupervised learning**: clustering (K-Means, DBSCAN) + dimensionality reduction (PCA)  
  - **Reinforcement learning (RL)**: sequential decision-making, introduced through **Q-Learning**

- **Chapter 3 — Model engineering & optimization (~6h)**  
  Training, evaluation, and deployment-oriented concerns:
  - Training mechanisms & persistence (weights, freeze/reload)
  - Stability & generalization (overfitting, regularization)
  - Evaluation protocols (metrics, robustness checks before integration)
